  <p align="center">
    1. Find the URL bar when you are on the Blooket page accociated with the hack.<br>

    <br>
    <img width="1440" alt="Screen Shot 2022-05-06 at 7 55 50 PM" src="https://user-images.githubusercontent.com/101288516/167229046-c7b35aef-b362-47a8-80f8-cfeba4c1f6f3.png"><br>
